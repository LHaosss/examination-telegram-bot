# examination-telegram-bot

### init

```md
# 当你首次打开项目，需要运行下面命令进行初始化

`npm install`

# 命令运行完成后下载所有依赖
```

### formal commit

```md
# 你需要使用`npm run cz`进行规范化提交

# 如果你不这样进行commit的话，commit会中断

`npm run cz`
```

### development

```md
# 执行下面命令通过tsx运行ts文件并监听文件变化

`npm run dev`
```

### build

```md
# 通过执行下面命令build项目

`npm run build`

# 通过这个指令会在build文件夹下生成js文件，我们可以用node运行这些文件
```
